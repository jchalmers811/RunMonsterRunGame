Run Monster Run is a game where you play as a newly created Frankenstein's monster on the run from angry villagers. 
You have to run forward to avoid being tracked down by villagers and make it through the terrain and other groups of
villagers who are attacking you from all angles. The goal to is escape alive. 


Known Issues: 
AI that spawn from behind will occasionally get caught out of bounds if they player backtracks. 
 
If player is not quickly killed by AI then they will keep spawning potentially slowing the game.
